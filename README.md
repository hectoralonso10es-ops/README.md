# README.md
Este proyecto trata sobre hacer una red local para una asesoría con 6 trabajadores

La red tiene 6 ordenadores una impresora un switch y un router que da acceso a internet

Primero se ha pensado qué necesita la empresa para poder trabajar

Después se ha hecho un dibujo de la red donde todos los equipos están conectados al switch y el switch al router

También se han puesto direcciones IP a cada dispositivo dentro de la red 192.168.1.0/24

La red sirve para tener internet compartir archivos y usar una impresora entre todos

Se ha comprobado que funciona haciendo pruebas de conexión entre los equipos

El objetivo es que la red funcione bien y sea fácil de usar para la empresa
